# Routing Rules

Use this file to define which AI tool should handle which type of task.

Example:
- search -> Perplexity
- paper archive -> Zotero
- document analysis -> NotebookLM
- design -> Claude
- implementation -> Codex
- memory update -> Gemini
- code state -> GitHub

Rule of thumb:

- give each tool one clear job
- prefer shared memory over repeated prompting
